# Microsoft 365 Admin Lab – Helpdesk Simulation

## Project Type
Microsoft 365 Administration • IT Service Desk Simulation • Cloud Identity & Access Management • End-User Support

---

## Project Summary
This project simulates real-world IT service desk operations within a Microsoft 365 enterprise environment. It demonstrates practical administration, troubleshooting, and user support workflows including onboarding, access control, collaboration services, and incident resolution.

The lab follows structured ITIL-style troubleshooting practices commonly used in Tier 1 / Tier 2 support environments.

---

## Key Highlights
- End-to-end user lifecycle management (creation → configuration → troubleshooting → recovery)
- Realistic helpdesk incident simulation and resolution workflows
- Role-based access control using security groups
- Authentication and session management scenarios
- Microsoft Teams collaboration and guest access configuration
- OneDrive sync troubleshooting and recovery scenarios
- Structured documentation aligned with IT support practices

---

## Skills Demonstrated
- Microsoft 365 user and license administration
- Identity and access management fundamentals
- Incident-based troubleshooting and resolution
- SharePoint and group-based permission management
- Microsoft Teams administration (basic)
- OneDrive sync issue diagnosis
- IT support workflow documentation
- Cloud service administration fundamentals

---

## Implementation Breakdown

### 1. User Lifecycle Management
- User creation and configuration in Microsoft 365 Admin Center
- License assignment and activation
- Account status validation
- First login verification

### 2. Authentication & Security Management
- Password reset procedures
- Account unlock operations
- Session revocation for security control
- Login validation after administrative changes

### 3. Access Control & Permissions
- SharePoint access troubleshooting
- Security group validation and assignment
- Permission-based access restoration

### 4. Collaboration Services
- Guest user creation and management
- Microsoft Teams meeting participation
- External collaboration validation

### 5. Cloud Storage & Sync Management
- OneDrive sync issue simulation and troubleshooting
- File sync interruption scenarios
- Recovery and validation of synced data

---

## Troubleshooting Scenarios

Each scenario follows a structured IT support workflow:

**Identify Issue → Analyze Cause → Apply Fix → Validate Resolution**

- User cannot sign in after account changes
- SharePoint access denied due to missing permissions
- OneDrive sync failure due to network interruption
- Guest user unable to access Teams meeting
- Session invalidation after security changes
